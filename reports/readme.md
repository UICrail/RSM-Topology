Continuous Integration reports, automatically generated by workflows.

Concerning the robot report:

IAO0000115 is the OBO annotation property for a textual definition. ROBOT will issue a warning each time a definition is absent or if there are multiple definitions.

We are not following OBO, and use rdfs:comment instead of IAO0000115. The warnings issued by ROBOT concerning this property can therefore be ignored. The user shall however take care of the existence and unicity of definitions.
