Wiki contents exported by the “Docs (Wiki)” workflow:

- single markdown file(s) and optional `images/` subfolder
- PDF and LaTeX versions (generated via Pandoc)

Prerequisites (see `https://raw.githubusercontent.com/Airy59/OntoQA/main/docs/WIKI_SETUP.md`):

- enable GitHub Wiki in repository settings
- initialize the wiki with at least one page (otherwise the workflow will warn and generate placeholder output)
